# Ubiquitous Language

| Term | Meaning |
| --- | --- |
| `CreativeProject` | 一次用户创作项目，保存目标和最终产物索引。 |
| `ChainRun` | 一次图片或视频链路运行，必须拥有独立 namespace。 |
| `StageRun` | 链路内固定阶段的一次执行记录。 |
| `ReviewReport` | 阶段或最终产物的量化审核结果。 |
| `Artifact` | 图片、视频、prompt、报告等可追踪产物。 |
| `Capability` | 云端 HTTP adapter、MCP 或 skill 的能力元数据。 |
| `ApiSelectionSnapshot` | 链路启动时 selected key 和 FreeModelGate 的冻结证据。 |
| `FreeModelGate` | 所有模型调用前的免费额度硬门禁。 |
| `KnowledgeChunk` | RAG 可检索片段，必须绑定 namespace、stageCode 和 contentHash。 |
| `RetrievalRecord` | 一次 RAG 检索证据，记录 query、filter 和命中 chunk。 |
